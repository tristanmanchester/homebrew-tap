# Tristanmanchester Tap

## How do I install these formulae?

`brew install tristanmanchester/tap/<formula>`

Or `brew tap tristanmanchester/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tristanmanchester/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
